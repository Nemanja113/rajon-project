# Rajon — Информационный портал района

Клиент-серверное веб-приложение для управления данными административного района.

## Стек

- Серверная часть: PHP + PDO
- База данных: PostgreSQL
- Клиентская часть: HTML, CSS, Vanilla JavaScript

## Функциональность

- Регистрация и аутентификация пользователей;
- Управление районами, улицами, учреждениями и событиями;
- Квартиры для аренды с корзиной и бронированием;
- Разграничение прав: администратор / пользователь;
- Журналирование входов и выходов.

## Установка

1) Клонировать репозиторий в папку htdocs/rajon;
2) Создать базу данных rajon_db в PostgreSQL;
3) Выполнить SQL из database.sql;
4) Запустить через XAMPP.
